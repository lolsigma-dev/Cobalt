## What is Cobalt?

Cobalt is a Roblox internal executor that started as a private, close-sourced project shared only among friends.  
It was built purely for fun, experimentation, and learning within two weeks time. The environment has been stripped and changed.

The project has since been open sourced because it is no longer actively used and was sitting untouched locally.  
Updates may still happen occasionally, but there is no fixed roadmap or planned update schedule.

---

## Why Open Source?

- **Why open source?**  
  I open sourced this because the project is no longer used, and leaving it on my computer untouched serves no purpose.  
  Sharing it allows others to learn from it, reference it, or build upon it.

- **Will this be updated?**  
  Possibly, but only occasionally. There is no active development plan. Expect rare updates if any at all.

## Features

- 28 UNC functions implemented  
- Full Luau execution  
- Level 8 identity  
- Full file system library  
- Input library (mouse / keyboard)  
- Stable execution 
- Stable TP Handler
- Working Auto-Execution
- Vectored-ExceptionHandler
- TCP Communications
- Fixed Namecall hooks
